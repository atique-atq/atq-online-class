Bootstrap provides UI (user interface) kit CSS framework and on the other hand Tailwind provides utility first CSS framework. Where Bootstrap focuses on responsive UI kit, Tailwind comes with predesigned widgets that facilitate web developers to design a website from scratch. 12 column concept was first introduced in bootstrap. Though Bootstrap and Tailwind are being used vastly worldwide, still boostrap user are far more than tailwind users. Probably this is because Bootstrap was developed many years ago than tailwind.

"Box Model" is used when we are talking about layout and design. The CSS box model is a box that wraps around each HTML element. It consists of: actual content, padding, borders and margins.Every box consists of four parts (or areas). Defined by their respective - the content edge, padding edge, border edge and the margin edge.

Semantic HTML elements are those that clearly describe the meaning. The HTML semantics refers to the tags that gives meaning to an HTML page rather than just presentation of html elements. A semantic element perfectly gives the meaning to both the developers and the browsers. Such as, a <p> tag indicates that the enclosed text is a paragraph. This is both semantic and presentational because user understands what is a paragraphs and browsers know how to display them. It makes HTML more comprehensible by better defining the different sections and layout of web pages. Some semantic tags are: <header>, <main>, <footer>, <article>, <section> etc.
